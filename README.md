# Sheffler Learning Standards

Shared guidelines for AI-assisted learning projects. Used as a git submodule in kids' project repos.

This is a teaching-oriented framework, not an enterprise engineering process. Keep it light.

## What's here

- `TEACHING_GUIDELINES.md` — how the AI agent should teach and interact
- `SESSION_DELIVERABLE.md` — what must be produced every session (the most important doc)
- `AGENTS_BASE.md` — shared baseline agent instructions for project repos
- `templates/session-log.md` — template for session deliverables
- `templates/curriculum.md` — living project roadmap template
