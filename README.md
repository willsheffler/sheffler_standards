# sheffler_standards

Shared standards, templates, and playbooks for Sheffler project repos.

## Contents
- `AGENTS.base.md`
- `STANDARDS_VERSION`
- `docs/templates/`
- `docs/playbooks/`

## Usage
In a project repo with `scripts/sync-standards.sh`:

```bash
SHEFFLER_STANDARDS_DIR=/path/to/sheffler_standards scripts/sync-standards.sh
```
