# Sheffler Family Standards

Shared guidelines for AI agents working with the Sheffler family. Used as a git submodule across family members' workspaces.

## What's here

### Learning and projects
- `TEACHING_GUIDELINES.md` — how AI agents should teach and interact with the kids
- `SESSION_DELIVERABLE.md` — what must be produced every coding/learning session
- `KIDS_GAME_DEV_ORIENTATION.md` — current direction for kids' game-dev, session modes, chat, and evidence capture
- `KIDS_GAME_DEV_V1.md` — compact v1 product spec for session modes, chat behavior, Dad participation, evidence capture, and deliverable automation
- `KIDS_GAME_DEV_DOGFOOD_V1.md` — CAP-converged implementation cut line for the first game-dev shell dogfood build
- `AGENTS_BASE.md` — shared baseline agent instructions for project repos
- `templates/session-log.md` — template for session deliverables
- `templates/curriculum.md` — living project roadmap template

### Family context
- `family/FAMILY_CONTEXT.md` — who's who, shared context all agents should know
- `family/RULES_AND_BOUNDARIES.md` — family rules agents must follow
- `family/PERSONAL_ASSISTANT_GUIDELINES.md` — how agents should help with non-coding tasks
